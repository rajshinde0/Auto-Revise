================================================================================
                    CP_ANAS vs MAIN PROJECT - KEY DIFFERENCES
================================================================================

This document explains how the 'CP_anas' subfolder differs from the main
project in the CP directory.

================================================================================
                              OVERVIEW
================================================================================

MAIN PROJECT (Root CP Folder):
- Name: MCQ Quiz Application / Quiz Master
- Type: Server-rendered Flask application
- Architecture: Monolithic (single app.py with templates)

CP_ANAS SUBFOLDER:
- Name: AutoRevise (Flashcard Study System)
- Type: Decoupled Frontend-Backend application
- Architecture: Separated Frontend and Backend folders

================================================================================
                         DATABASE DIFFERENCES
================================================================================

MAIN PROJECT DATABASE (mcq_flashcards):
- 16 tables total
- Focus: Quiz results, achievements, user statistics
- Key tables: questions, results, achievements, user_achievements
- 46+ documented SQL queries
- Connection pooling implemented

CP_ANAS DATABASE:
- Expanded schema with flashcard support
- Key tables: decks, cards, card_performance (spaced repetition)
- MCQ categories table (10 categories)
- MCQ performance tracking per category
- Upload logging for admin bulk imports

================================================================================
                       TECHNOLOGY DIFFERENCES
================================================================================

MAIN PROJECT:
- Flask 3.0.0 with Jinja2 templating
- Server-side rendering
- bcrypt for password hashing
- Flask-WTF for CSRF protection
- Rotating file logging

CP_ANAS:
- Flask with Flask-CORS (API mode)
- Client-side rendering (vanilla JS)
- Fetch API for AJAX requests
- Papa Parse for CSV parsing (client-side)
- Font Awesome for icons

================================================================================
                           USE CASE DIFFERENCES
================================================================================

MAIN PROJECT - Best For:
✓ Traditional quiz-taking experience
✓ Timed assessments with immediate results
✓ Achievement-based motivation
✓ Simple deployment (single server)
✓ SEO-friendly (server-rendered pages)

CP_ANAS - Best For:
✓ Long-term learning with spaced repetition
✓ Creating custom study content (flashcards)
✓ Multiple content types (flashcards + MCQs)
✓ Modern SPA-like experience
✓ Potential for mobile app conversion

================================================================================
                              SUMMARY
================================================================================

The main difference is that the ROOT PROJECT is a QUIZ APPLICATION focused on
testing knowledge through timed MCQ sessions, while CP_ANAS is a LEARNING
MANAGEMENT SYSTEM focused on helping users LEARN and RETAIN information using
flashcards with the scientifically-proven SM-2 spaced repetition algorithm.

Both projects share:
- User authentication with bcrypt
- Achievement/gamification systems  
- MySQL database backend
- Flask framework
- Admin capabilities

But they serve different educational purposes:
- Main Project = ASSESSMENT (test what you know)
- CP_anas = LEARNING (build and retain knowledge)
